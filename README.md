**Prime Khaled Bot** is a powerful WhatsApp automation bot designed for groups, business, and entertainment.

## ⚙️ Setup Configuration

Create a `.env` file:

```env
BOT_NAME=Prime Khaled Bot
OWNER_NAME=Prime Khaled
PREFIX=.
OPENAI_API_KEY=your_api_key
```

---

## 📱 Pairing Code Login

1. Run the bot:
```bash
node index.js
```

2. Enter your WhatsApp number  
3. You will receive a **pairing code**
4. Go to WhatsApp:
   - Settings → Linked Devices → Link with code
5. Enter the code and connect
